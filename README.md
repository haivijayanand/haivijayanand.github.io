# haivijayanand.github.io

Personal technical blog of **K Vijay Anand**,   
Chennai, India.

Live site: [https://haivijayanand.github.io](https://haivijayanand.github.io)

---

## Topics

- Aero-mechanical systems and test rigs
- Vibration diagnostics and signal processing
- Data pipelines (Python, TDMS, HDF5, SQLite)
- Science visualisation
- Instrumentation and control

## Built With

- [Jekyll](https://jekyllrb.com/) static site generator
- [Chirpy](https://github.com/cotes2046/jekyll-theme-chirpy) theme
- Hosted on GitHub Pages (free)

## Local Development

```bash
gem install bundler
bundle install
bundle exec jekyll serve
# Visit http://localhost:4000
```
